About

warning
This project is a decentralized LXD server control panel, so you only need a single instance which can be hosted anywhere.

Features

Here are some of the features of the panel:

SPA control panel

The control panel and the rest server agent can be installed together on a single server, or on its own if you have multiple servers. You are not tied to logging into multiple panels on different servers as all communications between the panel and the server are done over rest with JWT authentication. The panel can be served as a static SPA (Single Page App) on staic host, or with SSR (Server Side Rendering) through nuxt.

Server Information:

Access server information like CPU, memory, disk, processes, network connections and logins.

Server Management:

Manage the server from rebooting, clearing memory cache and much more, extendable to suit your needs with custom tasks.

APIs:

Build custom restful API endpoints, which can be used by your applications with full access to the underlying system and the fat-free framework.

LXD:

Easily manage all aspects of the LXD server, including containers, certificates, images, profiles, networks, files, logs, metadata and operations.

Routes:

You can also manage web and port forwarding, simply add a domain and set where it should be reverse proxied to, including multiple upstream's and Let's Encrypt Certificates.

Tasks:

Tasks allow you to run custom or predefined system tasks on the server once or at specific intervals. Which is great for maintenance tasks or deployment triggers like git pulling.
